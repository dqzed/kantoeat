# Kanto Eat

A PHP + HTML + CSS + JavaScript school project for a school canteen menu system.

## Includes
- Landing page
- Login / Sign up
- Customer dashboard
- Food detail page
- Admin dashboard
- Session-based add/edit/delete demo actions

## Notes
- Database SQL script is provided as `kantoeat_db.sql`.

## Database Setup Instructions
1. Open XAMPP Control Panel and start Apache and MySQL.
2. Open your web browser and go to `http://localhost/phpmyadmin/`.
3. Go to the "Import" tab.
4. Click "Choose File" and select `kantoeat_db.sql` located in this project folder.
5. Scroll down and click "Import" to execute the script and set up the `kantoeat_db` database, tables, and seed data.
